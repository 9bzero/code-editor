# code-editor

In-browser code editor with syntax highlighting and live preview.

## Supported modes

- HTML / CSS / JavaScript (with live preview iframe)
- TypeScript
- JSON (with format on save)
- Markdown

## Features

- Syntax highlighting via CodeMirror
- Live preview for HTML/CSS/JS (reloads on each change)
- Console output panel
- Auto-indent and bracket matching
- Theme toggle (dark/light)
- File open/save (reads/writes local files via the File System Access API where supported)

## Run

```bash
npm install && npm run dev
```
